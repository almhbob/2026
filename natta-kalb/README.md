# نطة كلب | Natta Kalb

نسخة رقمية عصرية للعبة سودانية تراثية تشبه الشطرنج/الدامة، على لوحة 5×5، لكل لاعب 12 كلبًا. يموت كلب الخصم عندما يقفز فوقه كلب اللاعب إلى مربع خالٍ خلفه أفقيًا أو رأسيًا.

## المميزات

- لوحة 5×5 كاملة.
- 12 كلبًا لكل لاعب وخانة وسط فارغة.
- حركة عادية إلى مربع مجاور فارغ.
- قفز فوق كلب الخصم وإقصاؤه عند وجود خانة فارغة خلفه.
- منع القفز القطري.
- وضع اختياري: الأكل الإجباري.
- كشف الحركات المتاحة بصريًا.
- عداد القطع المتبقية لكل لاعب.
- تراجع عن آخر حركة.
- شاشة قواعد وهوية سودانية حديثة.

## التشغيل على Termux / Android

```bash
git clone https://github.com/almhbob/2026.git
cd 2026/natta-kalb
npm install
npx expo start --localhost --clear
```

ثم افتح الرابط في Expo Go.

## التشغيل من ملف ZIP

```bash
unzip natta-kalb.zip
cd natta-kalb
npm install
npx expo start --localhost --clear
```

## اختبار منطق اللعبة

```bash
npm test
```

## القواعد البرمجية المعتمدة

- الاتجاهات القانونية: أعلى، أسفل، يمين، يسار.
- الحركة العادية: خطوة واحدة إلى خانة فارغة.
- الأكل: قفزة من خانتين فوق قطعة خصم، بشرط أن تكون خانة الهبوط فارغة.
- الفوز: عند تصفير قطع الخصم أو منعه من أي حركة قانونية.
